## Loops while and For 

***QUE ES UN LOOP?***

Es un bucle

Proceso que se repite indefinidamente

Serie de instrucciones que se reproducen indefinidamente
mientras no se cumpla una instrucción previamente establecida

***LOOP***

Es una estructura de control que permite ejecutar 
un bloque de instrucciones de manera repetida mientras
se cumpla una condición específica o hasta que se alcance 
una estado determinado

***WHILE***

Los bucles while son Utiles para repetir
instrucciones mientras una condición sea verdadera. Son como sentencias
if que se repite.


**WHILE (condición boleana) {ejecuta este código si es true}**


**FOR**

Los bucles for son útiles para repetir instrucciones un
número determinado de veces.


***NESTED LOOPS***

Un loop dentro de otro loop.


***FRAMECOUNT***


Variable numérica que registra
la cantidad de fotogramas dibujados
desde que comenzo el boceto.
El valor de framecount es 0 dentro del setup()
se incrementa en 1 cada vez que finaliza la
acción del código en draw.
